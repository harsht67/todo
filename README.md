# Todo App 

## Features
- Add/Edit/Delete todo 
- Filter by active/completed/all 
- Change status of todo (active/completed)
- Clear completed todos
- Theme (light/dark)
- Fullstack application (MERN stack)

## Built with 
- React 
- Sass => for styling 
- Redux 
- Axios => to fetch data 
- Mongodb => database 
- Mongoose => for schema 
- Express => backend server
